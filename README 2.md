# Dealer Monthly Payment Calculator

This project is a simple web app for estimating monthly car loan payments based on the amount financed, APR, and loan term.

## What this app does

- Calculates the estimated amount financed
- Estimates monthly payments for common loan terms
- Provides a simple interface for learning and testing financing calculations

## Important note

This app is not free to use. It is intended for learning and educational purposes only.

You may download, review, and study this project to learn:
- React and Next.js
- component props and state management
- financial calculation logic in a real app

## Installation

Make sure you have Node.js and pnpm installed.

```bash
pnpm install
pnpm dev
